Deze site

Ik heb deze site gemaakt omdat dit moet voor een informatica project van school. De volledige css en html is goed gekeurd op de html en css W3C validator. Voor de beoordeling van deze site lees alstublieft de features pagina goed door, hierin worden alle eigenschappen van deze site in uitgelegd.

Eisen site

Hier beneden zie je de tabel die op moodle te vinden is met waar mijn site aan moet voldoen. In deze tabel heb ik zelf elk onderdeel aangevinkt wanneer ik aan de eisen voldeed, en een verwijzing naar de bijbehorende plekken waar deze onderdelen zijn te vinden in mijn site.
